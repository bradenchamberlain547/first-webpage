HTML

HTML stands for Hyper Text Markup Language. This is a tag based language where tags are created with greater than and less than signs.

For example (Check on the website after reading the code):

<p> is a paragraph to show on the website </p>

the code above is a different font because of the code vs p opener and closer

There are special characters such as & that have to be defined in a specific way in order to show it correctly on the website and not break the code.


                def fun(coding):
                    print(coding, "is fun")
                fun("HTML")

Whitespace is honored in HTML/CSS so when writing the code above, notice the IDE spacing is affecting where it shows up on the webpage. Fix this by removing the spacing and putting the code in the IDE on the far left side like below.

    print(coding, "is fun")

Inline vs Block

Inline means that is inline with the text. Block level forces a new line

Block Level:div, p, h1, h2, h3, h4, h5, h6, li

Inline elements are span, img, a, strong, em
